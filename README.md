# 📝 JotBar

**JotBar** is a minimalist, native macOS menu bar utility designed for rapid task management. Built with SwiftUI, it focuses on speed, simplicity, and visual urgency.


## ✨ Features

- **Menu Bar Native:** Lives in your system tray for instant access.
- **Visual Urgency:** Circular progress rings show exactly how much time is left before a deadline.
- **Smart Notifications:** Customizable reminders (5, 15, or 30 mins before) to keep you on track.
- **Minimalist UI:** Focus on what matters.
- **Launch at Login:** Option to have JotBar ready the moment you start your Mac.
